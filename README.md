# DataBook Development Guide

## First-time Set Up

```bash
cd backend/
# install packages
yarn
# migrate DB. You must have the MySQL DB server running
yarn migrate
```

## Running Dev Environment

```bash
cd backend/
yarn dev
```

You can also use the VSCode Node.js debugger by hitting `F5` when `app.js` is focused.

```bash
cd frontend/
yarn dev
```
